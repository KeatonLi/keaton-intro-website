# 相册使用说明

## 如何添加照片

将照片文件放入此目录，系统会自动扫描并展示。

## 文件名格式

建议使用以下格式命名照片，以便自动提取信息：

```
标题_分类_日期_描述.jpg
```

### 示例

- `深圳湾日出_travel_2024-01-15_清晨的深圳湾公园.jpg`
- `公司年会_work_2023-12-28_年度优秀员工.jpg`
- `周末咖啡_life_2024-02-20.jpg`

## 支持的分类

- `life` - 生活
- `travel` - 旅行
- `work` - 工作
- `tech` - 技术

## 支持的格式

- JPG / JPEG
- PNG
- WebP
- GIF

## 注意事项

1. 建议图片尺寸不小于 800px，以保证显示效果
2. 文件名中的中文需要 URL 编码或直接使用中文
3. 日期格式为 `YYYY-MM-DD`
4. 照片会自动按日期倒序排列（最新的在前面）
