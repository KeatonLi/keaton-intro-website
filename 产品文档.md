# Keaton 个人网站 - 产品文档

## 📋 项目概述

这是一个基于 Vue.js 构建的个人作品集网站，专为 Java 工程师 Keaton 设计。网站采用现代化的设计理念，以蓝色为主色调，展现简洁优雅的视觉效果。项目集成了个人介绍、作品展示、技术博客等多项功能，旨在全面展示个人的专业技能和项目经验。

## 🎯 产品定位

- **目标用户**: 潜在雇主、合作伙伴、技术同行
- **核心价值**: 专业形象展示、技术能力证明、个人品牌建设
- **使用场景**: 求职面试、项目合作、技术交流、个人品牌推广

## ✨ 核心功能

### 1. 个人介绍模块
- **首页展示**: 个人简介、职业理念、技术追求
- **技术栈展示**: 后端技术、数据库、前端技术、开发工具
- **核心价值观**: 创新思维、工程实践、团队合作

### 2. 关于我页面
- **技术专长**: Java 企业级应用开发、微服务架构设计
- **工作经验**: 高并发系统架构、代码质量优化、团队协作
- **个人故事**: 学习之路、职业发展、技术成长历程
- **技能评估**: 可视化技能水平展示
- **联系方式**: 多渠道联系信息

### 3. 作品集展示
- **技术项目**: 
  - 企业管理系统 (Spring Boot + MySQL + Vue.js)
  - 电商平台后端 (Spring Cloud + Redis + RabbitMQ)
  - 数据分析平台 (Apache Kafka + Elasticsearch + Docker)
- **工具库开发**: 通用数据校验、Redis 缓存封装、API 文档生成
- **技术博客**: Spring Boot 最佳实践、微服务架构设计、性能优化技巧
- **工作瞬间**: 图片展示个人工作和生活状态
- **成就展示**: 技术团队负责人、性能优化专家、微服务架构实践

### 4. 技术博客系统
- **文档风格设计**: 类似开源文档网站的阅读体验
- **Markdown 渲染**: 支持完整的 Markdown 语法和代码高亮
- **搜索功能**: 实时搜索文档标题和内容
- **统计信息**: 文档数量、总字数、技术标签统计
- **响应式布局**: 左侧目录导航 + 右侧内容展示

## 🛠️ 技术架构

### 前端技术栈
- **框架**: Vue 3 (Composition API)
- **路由**: Vue Router 4
- **构建工具**: Vite
- **样式**: 原生 CSS3 (蓝色主题设计)
- **Markdown 处理**: markdown-it + Prism.js

## 🚀 部署方案

### 目标平台
- **主要平台**: Vercel
- **备选方案**: Netlify, GitHub Pages

### 部署流程
1. 代码推送到 GitHub 仓库
2. 连接 Vercel 进行自动部署
3. 配置自定义域名（可选）
4. 启用 HTTPS 和 CDN 加速

### 环境配置
- Node.js 版本: 16+
- 构建命令: `npm run build`
- 输出目录: `dist`
- 部署分支: `main`

## 🔧 开发指南

### 本地开发
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

### 内容管理
- **博客文章**: 在 `src/blog/posts/` 目录下添加 Markdown 文件
- **项目信息**: 在对应的 Vue 组件中修改数据
- **样式调整**: 在 `src/style.css` 中修改全局样式
- **路由配置**: 在 `src/router/index.js` 中管理路由
